About

Heartify is a clean and professional site template, perfect for all types of companies and personal web sites. Built on the Advanced Bootstrap 3 framework with HTML5 / CSS3. Its responsive too - Will fit great on all devices. More features like 5 Homepages, 2 Portfolio pages, About page, Services page, Masakini layouts, Timeline homepages and much more gives you more space to work with and also with a few lines of code.

Clean coded and Easy updation are the key features. Go ahead to know more.

CSS files and structure

I have used Ten CSS files in this theme.

  1. style.css - Contains all of the specific stylings for the page.
  2. bootstrap.css - Bootstrap css.
  3. font-awesome.min.css - Fontawesome Icons.
  4. owl.carousel.css - Contains all of the specific stylings for Owl Carousel
  5. owl.theme.css - Contains all of the specific stylings for Owl Carousel
  6. owl.transitions.css - Contains all of the specific stylings for Owl Carousel
  7. bootstrap-datetimepicker.css - Contains all of the specific stylings for Datepicker
  8. responsive-styling.css - Contains all of the specific stylings for responsive styling
  9. layerslider.css - Contains all of the specific stylings for Layerslider Plugin
  10. isotope.css - Contains all of the specific stylings for Isotope plugin

Javascript

This theme imports Fifteen Javascript files.

  1. jquery.js - To Enable jQuery.
  2. bootstrap.js - Plugin for Bootstrap framework.
  3. html5shiv.js - Enable HTML5.
  4. layerslider.kreaturamedia.jquery.js - Layer slider plugin.
  5. layerslider.transitions.js - Layer slider plugin.
  6. greensock.js - Layer slider plugin.
  7. bootstrap-datetimepicker.js - Datepicker plugin.
  8. jquery.isotope.min.js - Isotope Plugin.
  9. stellar.js - Parallax plugin.
  10. owl.carousel.js - Owl Carousel plugin.
  11. waypoints.js - Waypoints plugin.
  12. waypoints-sticky.js - Waypoints plugin.
  13. respond.js - IE mediaqueries plugin.
  14. stellar.js - Parallax plugin.
  15. custom-scripts.js - Theme custom plugin.
